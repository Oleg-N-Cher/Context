TinyContext
A tiny programming language written in itself
Andrei V. Khokhlov  - https://www.avhohlov.narod.ru/p1800en.htm

Data types: char, byte, word
Keywords: begin, end, if, inline, return, while
Function calls, description of variables. See the source code of the compiler for details

Currently there is a compiler for Windows and running directly on AgonLight,
capable of generating code for Z80 compatibility mode (ADL=0).

Development and support for Agon, ZX Spectrum and BK-0010 is handled by
Oleg N. Cher - https://github.com/Oleg-N-Cher/Context


TinyContext
Крошечный язык программирования, написанный на самом себе
Андрей Владимирович Хохлов - https://www.avhohlov.narod.ru/p1800ru.htm

Типы данных: char, byte, word
Ключевые слова: begin, end, if, inline, return, while
Вызов функций, описание переменных. Подробнее смотрите исходники компилятора

На данный момент есть компилятор для Windows и работающий непосредственно
на самом AgonLight, способный генерировать код для режима совместимости
с Z80 (ADL=0).

Разработкой и поддержкой для Agon, ZX Spectrum и БК-0010 занимается
Oleg N. Cher - https://github.com/Oleg-N-Cher/Context
